Fixing a section/div on the top of a website on scroll.
Suppose we want to fix a div at the top when scrolltop position is at 572.
Html:
<div class="box"></div>
Css:
.fixed{
  position: fixed;
  top: 0px;
}
jQuery:
$(window).scroll(function(){
  if($(this).scrollTop()>572){
   $(".box").addClass("fixed");
  }else{
   $(".box").removeClass("fixed");
  }
});

Repeat animation every time an other button/text/link etc is clicked?
We are using 3 different classes for explaination:
1) .animation = For animation.
2) #animate-me = Id which we are going to animate.
3) .click-me = On clicking this we will trigger the animation.

if(!$("#animate_me").hasClass("animation")){
  $(".click-me").click(function(){
   $("#animate_me").addClass("animation");
  setTimeout(function(){
   $(".animate_me").removeClass("animation");
  }, 1000);
 });
};

Notes:
a) Notice the "!" in first line of the code if(!$("#animate_me").hasClass ("animation")); this "!" means the opposite of what our code is acutally saying i.e. if "#animate_me does not have class animation".
b) The "1000" at the end of the code represents the delay of 1 second.

Toggling up/down, +/- font awesome icons using jQuery?
Method 1:
  $("i", this).toggleClass("class-1 class-2");
Method 2:
  $(this).find("i").toggle();
Tip:
Method 2 is the best because it can be used with other elments as well.
Ex:-
Html:
<div id="xyz">
  <img src="down.png">
  <img src="up.png">
</div>
jQuery:
$("#xyz").click(function(){
  $(this).find("img").toggle();
});

function addEvent(obj, evt, fn) {
