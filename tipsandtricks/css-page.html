Horizontally center a fixed or absolute positioned element?
.class{
  position: fixed/absolute;
  left: 50%;
  transform: translate(-50%, 0);
}
Vertically & Horizontally center a fixed or absolute positioned element?
.class{
  position: fixed/absolute;
  left: 50%;
  top: 50%;
  width: 300px;
  height: 200px;
  margin: -100px 0px 0px -150px;
}

Note: give margin-top half the height in (-ve) and margin-left half the width in (-ve).

Fix footer at the bottom of the page using flexbox?
Html:
<div class="wrapper">
  <header></header>
  <main></main>
  <footer></footer>
</div>
Css:
html,body{
 height: 100%;
 width: 100%;
}
.wrapper{
 min-height: 100%;
 display: flex;
 flex-direction: column;
 align-items: stretch;
}
main{
  flex-grow: 1;
}
header, main, footer{
  flex-shrink: 0;
}
